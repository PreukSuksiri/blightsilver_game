# Card Ability Text Review

Generated review of `card_data.xlsx`. Priority: shortest clear English.

## Style rules (apply everywhere)

- Use **Heads:** / **Tails:** (not "if head").
- **Flip N coins** (plural).
- **both players**, **attacker has**, **foe chooses**.
- Drop redundant **Affinity** after "vs Nature".
- Prefer **unit** over **card** for characters; **cell** for grid squares.
- **ATK and DEF** (not ATK&DEF).
- **0-cost traps**, **Mutagen flag**, **Venom flag** (sentence case).
- **Immune to Tech/traps** instead of long "unaffected by".
- **Reckoning** = battle resolution phase (keep as-is).
