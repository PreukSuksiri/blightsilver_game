- Create full project structure
- Build 5x5 GridContainer
- Create reusable Card.tscn (front + hidden back)

## Day 2: Card System + Data
- CharacterData, TrapData, TechData resources
- Load example cards with affinities and special effects

- One-use Trap system
- Implement Character special effects

- Basic AI opponent
- Hot-seat 2-player mode

## Day 7: Menu + Save + Export
- Main Menu + Match Setup
- Save system
- Build for Windows + Android test

**Milestone:** Fully playable local 2-player + vs AI match with all core rules.
